## 2. Hintergrund der Arbeit

**Art der Arbeit:**
```
Masterarbeit
```

**Kontext:**
```
Masterarbeit an der TU Wien, Studiengang Medieninformatik.

Die Arbeit untersucht alternative Input-Methoden für VR-Anwendungen, die für
Menschen mit eingeschränkter Handmotorik zugänglich sind. Aktueller Standard
für VR-Interaktion ist Hand-Tracking, welches präzise Handbewegungen erfordert.
Für Nutzer mit Tremor, Cerebral Palsy oder anderen motorischen Einschränkungen
ist dies oft nicht nutzbar.

Die Arbeit entwickelt und evaluiert ein adaptives Input-System, das sich
automatisch an verschiedene Grade motorischer Einschränkungen anpasst.
```

**Hauptziel der Arbeit:**
```
Entwicklung und Evaluation eines adaptiven Input-Systems für VR, das sich
automatisch an verschiedene Grade motorischer Einschränkungen anpasst und
dabei gleichwertige Nutzererfahrung wie Standard-Hand-Tracking bietet.

Forschungsfragen:
1. Welche alternativen Input-Methoden sind für VR mit Einschränkungen geeignet?
2. Wie kann ein System automatisch den Grad der Einschränkung erkennen?
3. Welche Adaptionsstrategien führen zur besten User Experience?
```

## 3. Verwendete Methoden/Theorien

**Forschungsmethoden:**
```
- User-Centered Design Ansatz
- Qualitative semi-strukturierte Interviews mit Betroffenen (N=15)
- Prototyping & iterative Entwicklung
- Usability Testing mit verschiedenen Zielgruppen
- Vergleichsstudie: Eye-Tracking vs. Voice Control vs. Gesture Control
- Quantitative Metriken: Task Completion Time, Error Rate, NASA-TLX
```

**Theoretischer Rahmen:**
```
- Universal Design Principles (Mace et al., 1997)
- ISO 9241-11 (Ergonomie der Mensch-System-Interaktion)
- WCAG 2.1 & EN 301 549 (Web/Digital Accessibility Guidelines)
- Ability-Based Design Framework (Wobbrock et al., 2011)
```

**Technologien/Tools:**
```
- Unity 3D 2023.2 für VR-Entwicklung
- Oculus Quest 3 als primäre Zielplattform
- Tobii Eye-Tracking Integration
- Azure Speech Services für Voice Control
- Custom Gesture Recognition (MediaPipe)
```

```

```

**Begründung:**
```
VR-Technologie hat sich 2019-2020 stark weiterentwickelt (Oculus Quest,
standalone VR, Hand-Tracking Mainstream). Papers vor 2019 beziehen sich oft
auf veraltete Hardware (tethered VR, externe Sensoren) und sind für moderne
Accessibility-Ansätze weniger relevant.

Ausnahme: Seminal theoretical papers (Ability-Based Design) bleiben relevant.
```
